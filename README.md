# Bar Landing Page - React + GSAP

A modern, fully responsive landing page for a bar, built with React and GSAP. The design focuses on smooth animations, elegant transitions, and a sleek visual identity to create an immersive experience.

https://velvet-pour-ytj1.onrender.com/

## Features
- Sleek Design – Modern layout with attention to typography, spacing, and visual hierarchy.

- Smooth Animations – Powered by GSAP, including scroll-triggered reveals, staggered text animations, and an interactive video background synced with scrolling.

- Fully Responsive – Optimized for all screen sizes, from mobile to desktop.

- Fast Performance – Built with React and optimized assets.
